# Crimson Bazaar — Multi-Vendor Marketplace

A bold, dark-themed multi-vendor ecommerce frontend built with **React + Vite + TypeScript + Tailwind CSS**.

## Features
- 🎨 Midnight Crimson theme — navy/black with red accents, glow hovers, smooth animations
- 🛍️ Home, Shop with filters/sort, Product Detail, Cart, Checkout, Order Success
- 🏬 Multi-vendor: vendor list, vendor stores, cart grouped by vendor
- 💖 Wishlist + 🔐 Mock auth — persisted via localStorage
- 📊 Vendor Dashboard (UI only)
- 📱 Fully responsive

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:8080

## Build
```bash
npm run build
npm run preview
```

## Stack
React 18 · Vite 5 · TypeScript · Tailwind CSS · shadcn/ui · React Router · Lucide
